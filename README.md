Musée de l'Illusion Paris - Projet SAE
Présentation du projet
Ce projet a été réalisé dans le cadre d'une SAE (Situation d'Apprentissage et d'Évaluation) en équipe au sein du BUT Informatique. L'objectif était de concevoir une stratégie digitale et une expérience immersive pour le Musée de l'Illusion de Paris.

Après une phase d'immersion (visite du musée et analyse de l'événement "Nocturnes 2026"), notre équipe a imaginé, scénarisé et promu une édition futuriste : Le Grand Bal des Illusions 2027.

Développement Technique
J'ai été responsable de l'intégralité de la conception technique et du développement de la plateforme web.

Architecture : Développement d'une Single Page Application (SPA) avec React (Vite).

Interface & Design : Création d'une identité visuelle premium (Gold & Dark) avec Tailwind CSS.

UX Immersive :

Système de particules et rayons de lumière dynamiques.

Animations au défilement via Intersection Observer.

Slider interactif sur-mesure (compatible souris et tactile).

Intégration et animation de composants SVG complexes (rideaux, lustres).

Internationalisation (i18n) : Mise en place d'un système bilingue (FR/EN) via React Context API avec persistance du choix via le localStorage.

Technologies utilisées
Framework : React (Vite.js)

Styling : Tailwind CSS

Navigation : React Router

Gestion d'état : Context API
